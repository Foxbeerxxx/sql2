2. `фамилия и имя сотрудника из этого магазина`
```
SELECT
    e.first_name,
    e.last_name
FROM
    staff e
WHERE
    e.store_id IN (
        SELECT
            s.store_id
        FROM
            store s
        JOIN
            customer c ON s.store_id = c.store_id
        GROUP BY
            s.store_id
        HAVING
            COUNT(c.customer_id) > 300
    );

```
